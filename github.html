Was ist GitHub?

GitHub ist eine Plattform, die auf Git aufbaut und Entwicklern ermöglicht, ihre Projekte online zu speichern und mit anderen zu teilen. Es erleichtert die Zusammenarbeit, das Management von Open-Source-Projekten und bietet Funktionen wie Issues und Pull Requests.
