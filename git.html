Was ist Git?

Git ist ein verteiltes Versionskontrollsystem, das Entwicklern hilft, Änderungen an Projekten nachzuverfolgen und mit anderen im Team zusammenzuarbeiten. Mit Git können verschiedene Versionen von Dateien gespeichert und bei Bedarf wiederhergestellt werden.
